Project 2

Chekhov

On the Chekhov website, we see a page split into two main sections that contain content of equal importance. By placing the two parts side by side, the designer creates contrast, grabbing more attention from the user, and the combination of animation and color changes creates dynamics.

Project 3

Timothee Roussilhe

Timоthee Roussilhe is a resume site for designer Timothy Roussilhe. The site is quite simple but creative at the same time. It uses a parallax effect to help attract visitors. As you scroll down, blocks appear that move around the screen. Interestingly, the blocks themselves also use parallax effects, which, when scrolling, create the feeling of watching a movie.
